Block And Inline Elements

By: Katie and Nick

Block Elements

  • Starts on a new line and takes up the entire line Examples: headings, paragraphs, lists and divs
  • Examples: < header > < / header>, < div > < / div >, < h1 > < / h1 > (ALL H TAGS)

Inline Elements

  • Dont start on a new line and only take up as much width as necessary and used within block line elements for styling purposes.
  • Examples: < a > < / a >, < strong > < / strong >, < span > < / span >

Void Elements

  • They are self closing elements that do not have a closing tag and are used to insert something into the page.
  • Examples: < break />, < img />, < base />, < meta />

Span and Div Elements

  • They are both containers, Div is a block element and span is an inline element. Div is used to group
    stuff
    together while span is used to style stuff and apply scripts.
  • Examples: < span > < span />, < div > < div />
